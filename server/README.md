# Sync Server (устарело)

Синхронизация Aprel/Matrix теперь использует **Vibix WebSocket** (`wss://sync.videoframe2.com`) — тот же сервер, что и для совместного просмотра Vibix.

Локальный сервер из этой папки **больше не нужен** для работы расширения. Оставлен только как справочная реализация протокола.

## Если всё же нужен локальный сервер

```bash
npm install
npm start
```

Порт по умолчанию: `3921`.
